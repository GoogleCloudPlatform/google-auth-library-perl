# Plan: Google Auth Library Perl Port - Clean-ups and Coro Optimization (2026-03-09)

## Goal
Optimize the Perl port of the Google Auth Library for cooperative concurrency (Coro) and align its architecture with the Python and Ruby implementations. This involves removing blocking primitives, decoupling the HTTP transport, and cleaning up dependencies.

## Tasks

### 1. Pluggable Transport Interface
- [x] Create `lib/Google/Auth/Transport.pm` as a base interface/documentation.
- [x] Implement `lib/Google/Auth/Transport/LWP.pm` as the default transport using `LWP::UserAgent`.
- [x] Ensure the transport interface mimics Python's `google.auth.transport.Request` (takes URL, method, headers, body; returns status, headers, body).

### 2. Refactor ID Token Key Sources
- [x] Break apart `lib/Google/Auth/IDTokens/KeySources.pm` into:
    - `lib/Google/Auth/IDTokens/KeySource/Http.pm`
    - `lib/Google/Auth/IDTokens/KeySource/Static.pm`
- [x] Modify `KeySource` classes to accept a `$transport` object instead of hardcoding `LWP::UserAgent`.
- [x] Update `lib/Google/Auth/IDTokens/Verifier.pm` to pass the transport through to the key source.

### 3. Remove Blocking Primitives
- [x] Audit the codebase for `Thread::Semaphore` or other `Mutex`-like objects.
- [x] Remove them; in a `Coro` environment, cooperative multitasking handles state safety between I/O points without needing OS-level locks.

### 4. Replace DateTime with Time::Piece/Epochs
- [x] Identify all usages of `DateTime`.
- [x] Replace with `Time::Piece` (core module) or simple epoch integers (`time()`) to reduce overhead and dependencies.

### 5. Test Suite Modernization
- [x] Refactor `t/key_sources_test.t` to use mocks for HTTP responses.
- [x] Remove dependency on `Crypt::OpenSSL::CA` for dynamic certificate generation in tests; use pre-generated static PEM strings or a simpler mock.
- [x] Ensure all tests can run under `prove -j$(nproc)`.

## Success Criteria
- [ ] `Coro::LWP` can be used without blocking the entire process.
- [ ] No hardcoded `LWP::UserAgent` in auth logic.
- [ ] Tests pass without `Crypt::OpenSSL::CA`.
- [ ] `DateTime` is removed from the dependency tree.
